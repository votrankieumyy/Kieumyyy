# 🚀 BlueStacks 5 FPS Optimizer v4.5 ELITE

**Ultimate BlueStacks Performance Optimization Tool - Maximize FPS & Minimize Input Latency**

## ✨ Features

✅ **5 Optimization Modules**:
  - 🚀 **FPS Optimization** - Windows system-level tweaks
  - ⚙️ **Deep Config** - C:\ProgramData\BlueStacks_nxt optimization
  - 🔍 **Install Optimize** - C:\Program Files\BlueStacks_nxt tuning
  - ⚡ **Low Latency Input** - File-based input optimization
  - 💥 **CMD Latency** - PowerCfg & Registry-based optimization

✅ **Beautiful UI** - Dark theme with color-coded buttons
✅ **Real-time Logs** - See optimization progress
✅ **One-Click Solution** - No manual configuration needed

## 🌎 System Requirements

- Windows 10 / Windows 11
- BlueStacks 5 installed
- .NET 6.0 Runtime (net6.0-windows)
- Administrator privileges (recommended)

## 🚀 Installation & Usage

### Method 1: Clone & Build

```bash
git clone https://github.com/votrankieumyy/Kieumyyy.git
cd Kieumyyy/BlueStacksFpsOptimizer
dotnet build -c Release
dotnet run
```

### Method 2: Run Compiled Executable

```bash
.\bin\Release\net6.0-windows\BlueStacksFpsOptimizer.exe
```

## 🙋 Troubleshooting

### FPS Still Low
- Update GPU drivers
- Close background applications
- Reduce game graphics settings
- Check CPU/GPU temps

### Input Latency High
- Run CMD LATENCY optimization
- Check USB connections
- Disable background input devices
- Verify polling rate settings

### BlueStacks Won't Start
- Restore config files
- Reinstall BlueStacks
- Check Windows event logs
- Verify GPU support
